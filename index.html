Kuidas luua kalender Microsoft Project'is

See leht aitab sul samm-sammult luua ja seadistada kalendri oma projektile.

Alusta õppimist

1. Uue kalendri loomine

Kõigepealt ava Microsoft Project ja vali ülemisest menüüst Project > Change Working Time. Seal näed vaikimisi kalendreid. Kui tahad teha oma kalendri, klõpsa nupul Create New Calendar.
