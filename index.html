Kuidas luua kalender Microsoft Project'is

Juhend samm-sammult

1. Uue kalendri loomine

Ava Microsoft Project ja vali menüüst Project → Change Working Time. Seejärel vajuta Create New Calendar, vali "Create new base calendar" ja pane kalendrile nimi.

2. Tööpäevade muutmine

Vali kalender ja muuda tööpäevi. Näiteks saad määrata laupäeva tööpäevaks või pühapäeva vabaks päevaks.

3. Tööaja muutmine

Vali tööpäev ja vajuta Details. Muuda tööaega näiteks 10:00–18:00.

4. Eripäevade lisamine

Ava vahekaart Exceptions ja lisa näiteks puhkus või püha. Sisesta kuupäev ja nimetus.

5. Kalendri kasutamine projektis

Mine Project → Project Information ja vali oma loodud kalender.

Kuidas kalender mõjutab ajakava

Kalender määrab, millal tööd saab teha. Kui päev on märgitud mitte-tööpäevaks, siis ülesanded ei liigu edasi. Samuti mõjutab tööaeg projekti kestust.
